# Content rating questionnaire (IARC) — answers

**Play Console → Monitor and improve → Policy → App content → Content ratings → Start questionnaire.**

## Answers

| Question | Answer | Why |
|---|---|---|
| Violence — realistic or graphic violence toward characters? | **No** | Machines break down; no character is harmed. A "strike" is a labor stoppage, not violence. |
| Cartoon / fantasy violence? | **No** | None. |
| Sexual content or nudity? | **No** | None. |
| Profanity or crude humor? | **No** | None. |
| **References to alcohol, tobacco, or drugs?** | **Yes → alcohol reference** | One playable plant is a brewery (Ironline Brewery & Bottling Works) that produces and bottles beer. Production only — no character drinks, no glamorized use. Declare it honestly; hiding it risks a rating strike. |
| Simulated gambling / real-money gambling? | **No** | None. |
| Horror / fear themes? | **No** | None. |
| Does the app allow users to interact or exchange content (chat, UGC)? | **No** | No chat, no shared content. Cloud saves are private per account. |
| Does the app share the user's current location with other users? | **No** | Location is never accessed at all. |
| Does the app allow users to make digital purchases? | **No** (today) | No Play Billing, no purchases. The premium plants are a free preview. **Re-run this questionnaire when the store ships.** |
| Is the app itself a web browser or search engine? | **No** | It's a game (TWA shell around its own content only). |

## Expected result

With only an "alcohol reference" flag, expect ratings in the range of:
**ESRB E / E10+ (Alcohol Reference)** · **PEGI 3–7** · **USK 0–6** · IARC generic 3+.
Any of these is fine for the listing and does not restrict distribution.
